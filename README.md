## Hunter Survivors - README

Gênero: Roguelike, Autoshooter, Bullet Heaven Engine: Godot (GDScript)

Estilo visual: 2D top-down 

Música e SFX: 8/16-bit 

Inspirações: Vampire
Survivors, Brotato, Megabonk

Descrição: Hunter Survivors é um autoshooter top-down focado em
sobrevivência. O jogador enfrenta hordas crescentes de inimigos enquanto
coleta XP para evoluir armas e habilidades. O objetivo é sobreviver o
máximo possível e acumular pontos.

Loop de gameplay: 

- Navegar pelo mapa
- Derrotar inimigos para obter XP
- Evoluir habilidades e armas
- Desviar de perigos e armadilhas
- Sobreviver pelo máximo de tempo possível

Condições: 
- Vitória: não existe — o objetivo é pontuar
- Derrota: HP chega a 0

Sistemas principais: 
- Dano por colisão com inimigos, projéteis e
armadilhas
- Pontuação baseada no tempo - Mapa com objetos estáticos e
interativos
- XP e evolução de personagem
- Progressão de dificuldade

Como rodar: 
1. Baixe o repositório
2.  Abra o projeto na Godot
3. Executea cena principal (Main.tscn)
